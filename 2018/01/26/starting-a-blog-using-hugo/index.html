Starting a blog using Hugo

I thought it would be fun to also start my own blog to document things that I have learned and am curious about and to recap the many conferences that I attend…and possibly other random things! So I decided to create a static blog with Hugo, the open-source static site generator and my experience with it has been very smooth and painless. It is very quick to set up and there are many pre-made themes to choose from.

I was aware of the plethora of static site generators out there and the more popular Jekyll, which seems to be natively supported by GitHub Pages. But, in the end, I didn’t want to succumb to choice paralysis nor did I want to spend too much time investigating and evaluating too many options. I had heard good things about Hugo, am fairly interested in Golang, and it seemed like it would be a good choice in the long run because of its fast build time. It also fulfilled my need for something simple that would get me into the rhythmn of blogging.

Overall, I would recommend it to anyone who wants to quickly spin up a quick blog. :)
